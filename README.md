# Weight Converter

This is a simple Python program that converts body weight between kilograms and pounds.

## Features
- Convert kilograms to pounds
- Convert pounds to kilograms
- Simple command line interface
- Beginner friendly Python project

## How It Works
The user enters their weight and specifies whether the weight is in kilograms (K) or pounds (L).  
The program then converts the value into the other unit and displays the result.

## Example

Enter your Body Weight: 70  
Is this weight in (K)g or (L)bs? K  

Output:  
Your weight in pounds is 154.35 lbs
